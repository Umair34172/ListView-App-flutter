# Counter App

A simple Flutter application that increments and decrements a counter.

## Features
- Increase counter
- Decrease counter
- Clean UI

## Technologies
- Flutter
- Dart

## How to Run
1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`
